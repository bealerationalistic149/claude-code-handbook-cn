## 🧰 常见问题

### 下载后找不到文件怎么办

先打开浏览器的下载记录，再点“在文件夹中显示”。也可以去 Windows 的“下载”文件夹查看。

### 双击后没有反应怎么办

先确认你下载的是完整文件。然后：

- 重新解压
- 右键选择“以管理员身份运行”
- 检查是否被杀毒软件拦截
- 再下载一次最新文件

### 文件打不开怎么办

先确认文件类型：

- `.exe`：直接运行
- `.zip`：先解压
- `.pdf`：用阅读器打开
- 网页文件：用浏览器打开

### 中文显示不正常怎么办

如果字体很小或显示异常，可以：

- 调大系统缩放
- 放大浏览器页面
- 更换字体查看
- 检查文件是否完整下载
